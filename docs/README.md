# bizforge — Documentation

Business operations and compliance reference files.

> Company-specific config comes from `.env`. The default company name is set via `COMPANY_NAME`.

## Directory Map

- `legal_templates/` — Generated SOWs and filled legal documents
- `brand_research/` — Saved brand screening reports
- `business/` — Reference:
  - `company-formation.md` — Steps to register the company
  - `tax-strategy.md` — UK tax optimization guide
  - `app-store-publishing.md` — Apple App Store pipeline
